# CapsPlus

Переключение раскладки на CapsLock (не отключает системное переключение)

## Зачем

- Чтобы использовать CapsLock как удобную глобальную клавишу смены раскладки вместо стандартных сочетаний.
- Чтобы уменьшить случайные срабатывания CapsLock и избавиться от «залипшего» верхнего регистра при наборе текста. 

## Что делает

- CapsLock → эмуляция Alt+Shift (переключает язык ввода).
- Shift+CapsLock → включает CapsLock в традиционном режиме.
- Ctrl+CapsLock → включает или отключает работу утилиты (Caps+ ON/OFF).
- В системном трее появляется иконка с подсказкой по состоянию и горячей клавише.

## Использование

- Запустите `CapsPlus.exe` (ничего дополнительно настраивать не нужно).
- Убедитесь, что в трее появилась иконка Caps+.
- Наведите курсор на иконку, чтобы увидеть состояние:
  - `Caps+: ON (Ctrl+CapsLock)`
  - `Caps+: OFF (Ctrl+CapsLock)`
- Правый клик по иконке:
  - `Enable Caps+` — включить/выключить обработку CapsLock.
  - `Exit` — закрыть утилиту.

## Дальнейшие планы

- Опциональная автозагрузка вместе с системой (например, через планировщик заданий).
- Ручное или автоматическое скрытие иконки из трея.
- Эксперименты с Alt+Caps:
  - смена языков с шагом в два при трёх и более раскладках;
  - принудительный выбор английской раскладки.
- Возможное объединение с отдельным проектом по переработке клавиши NumLock.

## Структура для самостоятельной сборки

- `src/caps_lang.c` — исходный код.
- `build.bat` — скрипт сборки.
- `icons/capslock.ico` — иконка для трея (опционально).

Для самостоятельной сборки нужен gcc для Windows (например, MinGW).  
Можно использовать `build.bat` или выполнить в командной строке:

```bash
gcc src\caps_lang.c -luser32 -mwindows -o CapsPlus.exe
```
